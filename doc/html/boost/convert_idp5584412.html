Function template convert

boost::convert — The main Boost.Convert deployment interface with algorithms.

Synopsis

// In header: <boost/convert.hpp>


template<typename TypeOut, typename TypeIn, typename Converter> 
  enable_if< cnv::is_cnv< Converter, TypeIn, TypeOut >, typename cnv::adapter< TypeOut, TypeIn, Converter > >::type 
  convert(Converter const & cnv);

Description

This the Boost.Convert main deployment interface with algorithms. For example, 

boost::array<char const*, 3> strs = {{ " 5", "0XF", "not an int" }};
std::vector<int>             ints;
boost::cnv::cstream           cnv;

cnv(std::hex)(std::skipws);

std::transform(
    strings.begin(),
    strings.end(),
    std::back_inserter(integers),
    boost::convert<int>(cnv).value_or(INT_MAX));
