Default Converter

The necessity to explicitly specify a converter as in 

int i = boost::convert("123", converter).value();

might be tedious or even error-prone if an application relies on certain consistent behavior associated with one particular converter. To accommodate such a scenario Boost.Convert introduces the concept of the default converter and deploys boost::cnv::by_default for the purpose.

There is no default converter set by default and, consequently, without additional configuration steps the following call will fail to compile: 

int i = boost::convert("123").value(); // No converter provided

After boost::cnv::by_default is defined, the following code compiles and deploys boost::cnv::lexical_cast when boost::convert() is called without an explicitly supplied converter: 

struct boost::cnv::by_default : public boost::cnv::lexical_cast {};


using std::string;
using boost::convert;


int    i = convert<int>("123").value();
string s = convert<string>(123).value();

// No explicit converter provided. boost::cnv::by_default is used.
BOOST_TEST(i == 123);
BOOST_TEST(s == "123");
